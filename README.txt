=== 오직예수로교회 홈페이지 배포 안내 ===

[배포 전 필수 설정]
1. Netlify 대시보드 → Site configuration → Environment variables
2. 새 변수 추가:
   - Key: CMS_ADMIN_PASSWORD
   - Value: 원하는 비밀번호 (예: ojc2024)

[배포 방법]
이 폴더 전체를 GitHub에 올리고 Netlify와 연결하세요.
(단순 드래그앤드롭은 Functions가 작동하지 않습니다)

[사용법]
1. 사이트 접속
2. 화면 하단 왼쪽 "CMS" 배지 또는 관리자 버튼 클릭
3. 설정한 비밀번호로 로그인
4. 내용 수정 → 저장 버튼

[이번주 말씀 자동 연결]
관리자 → 유튜브 링크 입력 → "이번주 말씀 자동 연결" 버튼
